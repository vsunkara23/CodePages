# CodePages

A Node.js wiki web application geared towards programmers to foster a community of driven 
developers who document code practices, software projects and the latest happenings 
in software.

Running Program:

    $ npm install 
    $ node index.js
   

Libraries/Npm Modules Used:

- [Markdown-it]
- [FlexSearch.js]
- [Uiud]
- [pug]
- [diff]
- [diff2html]
- [jquery]
- [Browsersync]
- [pm2]
